Anaconda

Download Anacondaâ€‹

  • For macOS: brew install --cask anaconda
  • For Linux/Windows: Install here.

Init conda because we don't have this commandâ€‹

./conda init zsh # The conda configuration is now in ~/.config/zsh/.zshrc
source $HOME/.config/zsh/.zshrc

Check the PATH with which -a pythonâ€‹

  • In Windows, $HOME/Anaconda3
  • In macOS with Intel chip, /usr/local/anaconda3/bin
  • In macOS with Apple chip, /opt/homebrew/anaconda3/bin
  • In Linux, $HOME/anaconda3/bin

Update condaâ€‹

conda update conda -y
conda update --all -y

nvim ~/.config/conda/cv.yml

Paste the following

name: CV
channels:
  - pytorch
  - conda-forge

  # PyTorch + CUDA

  # Others
  # - jupyter
  - opencv
  - matplotlib
  - notebook
  - tqdm

Run

Install formatter for jupyter notebookâ€‹

Auto activate base exampleâ€‹

# Store config in ~/.condarc

Create environment directlyâ€‹

# e.g.
# Create an env for Mkdocs deployment

# Activate this environment

# Now work with this cute MKDOCS env!
pip install mkdocs
pip install mkdocs-material

Remove an environmentâ€‹
